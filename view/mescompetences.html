    Mon compétences

    Front-end

    • HTML :
      Langage utilisé pour structurer le contenu d'une page web (titres, paragraphes, images, liens, etc.).

    • CSS :
      Langage de style utilisé pour définir l'apparence d'une page web (couleurs, polices, mises en page, etc.).

    • JavaScript :
      Langage de programmation utilisé pour rendre une page web interactive et dynamique (animations, formulaires, manipulations d'éléments, etc.).

    Back-end

    • PHP :
      Langage de script serveur utilisé pour le développement web dynamique.

    • C :
      Langage de programmation bas niveau, rapide et efficace, souvent utilisé pour les systèmes et l’embarqué.

    • C++ :
      Extension de C avec la programmation orientée objet, utilisé pour les applications complexes et performantes.

    • Java :
      Langage orienté objet, portable et sécurisé, couramment utilisé pour les applications d’entreprise et mobiles.
