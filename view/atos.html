    • ATOS PROJECT

    This project consisted of redesigning the website of an IT service company (here Atos), so that it would be suitable for middle school students looking for a 9th grade internship in such a company. We had to simplify some texts and explanations about the company’s activity. The website also had to be easy to use.

    Objectives

    The means used to meet these objectives:

    Teamwork (group of 3)

    Use of collaborative tools

    Mockup creation

    We had to build the site as a team of 3. To ensure we were aligned on development, we created mockups using Whimsical. To stay in touch, we used Discord. To code collaboratively without bugs from overlapping work, we used GitHub.
